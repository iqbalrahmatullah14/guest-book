# 📖 Buku Tamu Kelas Open Source

Selamat datang di repositori proyek **Buku Tamu Kelas**! Ini adalah repositori simulasi yang digunakan sebagai wadah praktikum mata kuliah untuk belajar bagaimana berkontribusi di dunia _Open Source_.

## 🎯 Tujuan
Mengumpulkan daftar seluruh mahasiswa yang mengikuti praktikum ke dalam satu halaman website. Halaman ini akan di-*deploy* menggunakan **GitHub Pages**.

## 🛠️ Cara Berkontribusi
Karena ini adalah proyek _Open Source_, Anda tidak diizinkan untuk mengubah kode langsung di repositori ini. Silakan ikuti aturan kontribusi yang ada di dalam file `CONTRIBUTING.md`.
